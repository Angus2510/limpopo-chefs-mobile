# App Store Screenshots Guide

## Required Screenshot Sizes for iOS:

### iPhone Screenshots (Required)

- **6.7" Display (iPhone 14 Pro Max):** 1290 x 2796 pixels
- **6.1" Display (iPhone 14):** 1179 x 2556 pixels
- **5.5" Display (iPhone 8 Plus):** 1242 x 2208 pixels

### iPad Screenshots (Optional but Recommended)

- **12.9" Display (iPad Pro):** 2048 x 2732 pixels
- **11" Display (iPad Air):** 1668 x 2388 pixels

## Screenshots to Take:

1. **Login Screen** - Shows the academy branding
2. **Dashboard** - Main screen with announcements preview
3. **Announcements** - Full announcements list with priorities
4. **Attendance** - Student attendance tracking
5. **Profile** - Student profile information

## Quick Screenshot Method:

### Using iPhone Simulator:

1. Open Xcode
2. Run iOS Simulator
3. Install the app (we can help with this)
4. Take screenshots using Cmd+S
5. Screenshots automatically saved to Desktop

### Using Real Device:

1. Install the current APK version
2. Take screenshots using device screenshot function
3. Transfer to computer for upload
